# bernstein-landing

Marketing site for [Bernstein](https://github.com/sipyourdrink-ltd/bernstein) — deterministic orchestrator for 30+ CLI AI coding agents.

Live at [**bernstein.run**](https://bernstein.run).

## Stack

Next.js · TypeScript · Tailwind CSS · Docker

## Develop

```bash
npm install
npm run dev
```

Then open <http://localhost:3000>.

## Deploy

`master` push triggers the GitHub Actions workflow that rebuilds the Docker image and rolls it on the production VPS.

## Brand

Source of truth for the Bernstein brand system lives at [`sipyourdrink-ltd/bernstein`](https://github.com/sipyourdrink-ltd/bernstein) and the parent studio brand at [sipyourdrink.ltd](https://sipyourdrink.ltd).
